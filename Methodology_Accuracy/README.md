# Methodology and Accuracy Metrics

This folder contains the complete methodology, experimental setup, evaluation metrics, and performance analysis of the proposed multimodal sentiment analysis framework.

It includes:
- Model architecture and pipeline design
- Cross-modal attention and Transformer fusion details
- Training configuration and hyperparameters
- Ablation study results
- Accuracy, F1-score, and MAE evaluations
- Confusion matrices and uncertainty analysis
- Performance comparison with baseline and existing models

The results are based on experiments conducted on the CMU-MOSI benchmark dataset using PyTorch.
